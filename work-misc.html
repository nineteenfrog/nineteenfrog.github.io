Misc. Previous Projects

DirectX 11 Engine

Created a game engine as a semester long project for my Foundations of Game Graphics Programming. The entire engine was developed by me using C++ and HLSL. Features include: PBR, Real-time shadows, Skybox, Blur Post Process. I might continue working on this in my free times and plan to implement cell-shading and a outline post-process. If you would like to download it this is the github page: https://github.com/nineteenfrog/DX11Engine

Finger Guns

Finger Guns was a project in which I worked with some of my friends. Finger Guns is a VR first-person shooter about a scientist who gets betrayed by the company who he developed deadly weapons for. Now, he must use those weapons to save himself...and maybe all of humanity. The main things I created while working in Finger Guns were the laser VFX for the hands and also the post-process shader. It was a combination of a cell shading technique and a cartoon outline effect. All of this was done using Unreal Engine 5.

Croak and Dagger

The video above was made by my friend who also worked on the project. I believe he speaks about the development and gives a good rundown on the game and presents it in an entertaining way too. This project was really fun to work in and we got a good working prototype going. I worked as a technical artist and created some shaders and vfx. I created a dissolve effect that would make the floating companion disappear once Croak (the frog) unsheathes his weapon. I also created the water shader using shifting normal maps. Lastly, I also created a tongue rig since we had planned to implement a tongue-slinging mechanic. Unfortunately, we didn't end up implementing it into the game but it was still fun to work on.
